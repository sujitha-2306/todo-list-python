# 📝 To-Do List CLI App (Python)

## 📌 Description

A simple and beginner-friendly command-line To-Do List application built using Python.
This app helps users manage their daily tasks efficiently with file storage support.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* 🗑️ Delete tasks
* 💾 Save tasks to a file
* 🔄 Automatically load tasks on startup

---

## 🛠️ Technologies Used

* Python
* File Handling

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/todo-list-python.git
```

2. Navigate to the project folder:

```bash
cd todo-list-python
```

3. Run the program:

```bash
python todo.py
```
